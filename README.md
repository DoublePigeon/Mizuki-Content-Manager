# A simple yet powerful manager that frees you from manually managing you Mizuki-blog contents

Mizuki CM is a WebUI application created with Gradio, and offers quick solution to managing items including pages as "About, Diary, Friends, Projects, Devices, Albums, Posts, Timeline, and skill"

# Usage

First install the requirements

```bash
pip install -r requirements.txt
```

Then run the .py file

```bash
python app.py
```

A URL will be shown in the terminal given it is booted successfully, simply open it in your browser and you can start managing your content!
